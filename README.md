Gen AI
This repository contains a simple Streamlit-based chatbot using Groq's LLaMA 3.1 70B model through LangChain.

1 Features
Streamlit UI
LLaMA-3.1-70B using Groq
API key is stored in .env
Simple question-answering chatbot
